# MediaWiki with LDAP

A MediaWiki installation with LDAP extensions.

## How to use this image

### Start a MediaWiki Server instance

To start a container use the following:
```sh
docker run \
	--name my-mediawiki-ldap \
	--detach \
	--publish 80:80 \
	pnwbob/docker-mediawiki-ldap
```
